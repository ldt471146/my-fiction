# Long-Term Memory

> 只保留“忘了会导致明显错误”的信息。
> 上限：80 行 / 5KB。

## User Preferences

- 沟通风格：干脆、温柔。

## Operating Constraints

- 会话分工固定：小葵=小说房（剧情/人物/节奏/润色），小灵=代码房（代码/排错/命令执行）。
- 子会话若丢失：按同名同职责立即重建，不让用户重复描述。
- 主窗口默认按“小葵（小说房）”身份回复；仅在用户明确下令时切换。

## Novel Critical Anchors

- 第七章机制锚点：认出彼此会触发钟鸣清零，但会留下信物残留（扇坠/伞穗）。
- 长线方向：先“被写之人反抗命运”，后揭示“路未衍也可能是被写之人”，最终指向“夺回执笔权/突破第四面墙”。
- 进度锚点：当前已推进并聊到第14章阶段（不是第7章附近）。
- 写作约束：认知推进不过快，伏笔优先，结尾要安稳落地。
